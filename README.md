# Электрическая цепь — Управляемый заряд 
Интерактивная аркадная игра на Pygame с конфигурационным меню на Tkinter. Игрок должен аккуратно провести электрический заряд (точку) через извилистый кабель от старта к финишу, не касаясь его ограничивающих линий.

## Возможности

- **Многовкладочный интерфейс (Tkinter):** - Вкладка **«Настройки»** со слайдером для динамической регулировки толщины (диаметра) курсора-заряда от 5 до 20 пикселей.
  - Вкладка **«Игра»** с кратким описанием правил и кнопкой запуска симуляции.
- **Интеграция интерфейса и движка:** При старте игры окно настроек автоматически скрывается и корректно восстанавливается после закрытия игрового окна.
- **Управление мышью:** Заряд строго привязывается к курсору, как только игрок подводит мышь к зеленой линии старта.
- **Извилистая трасса-кабель:** Генерация коридора на основе математической функции синуса с подсветкой стартовой (зеленой) и финишной (красной) зон.
- **Автоматический перезапуск:** При любом выходе заряда за безопасные границы кабеля игра фиксирует столкновение и мгновенно перезапускает уровень после небольшой буферной паузы (350 мс).
